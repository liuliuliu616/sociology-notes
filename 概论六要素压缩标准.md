# 概论笔记六要素压缩标准 v2.0

> 基于一平哥"概论笔记六要素"课程校准。本文件是后续所有章节压缩的统一执行标准。

---

## 一、要素标准化定义

### 六要素速查

| 序号 | 要素名 | 一句话 | 判定反问 |
|------|--------|--------|---------|
| ① | **内涵（定义）** | 这个东西是什么 | 能拆出属和种差吗？ |
| ② | **特征** | 它由哪些下位概念构成（放宽：有哪些属性/特点/指标） | 原文列出了子概念或罗列了属性吗？ |
| ③ | **外延（类型）** | 它在现实中有哪些分类 | 有分类标准和分类结果吗？ |
| ④ | **功能 / 原因与机制** | 它起什么作用，或什么导致了它 | 方向是概念→社会还是社会→概念？ |
| ⑤ | **视角** | 哪些学者怎么看待它 | 有名有姓吗？ |
| ⑥ | **变迁** | 它从过去到现在怎么变的 | 有时间对比吗？ |

### ① 内涵（定义）

**标准**：属 + 种差，缺一不可。

- "属" = 上位范畴（这个东西属于哪一类）
- "种差" = 它跟同类其他东西的关键区别

**判定**：原文有"XX是指…""所谓XX，就是…""XX定义为…"且能同时拆出属和种差 → 归内涵。只有描述性介绍、拆不出属+种差 → 不硬标定义，考虑归入其他要素或标缺口。

**例**：`社会化是个人习得社会角色和规范的过程` → "过程"是属，"习得角色和规范"是种差。✅

---

### ② 特征

**标准（一平哥原意）**：核心概念与亚概念（下位概念）之间的关系——核心概念太抽象，需要更低层次的子概念来展开解释。

**放宽规则**：教材不完全按"亚概念关系"组织时，以下也归特征：
1. 原文列出了下位概念/构成要素（严格匹配一平哥标准）
2. 原文用"第一…第二…"罗列的属性/特点
3. 原文明确说"XX具有以下特征/特点"引领的内容
4. 概念的测量指标/维度

**判定流程**：

```
原文内容
  │
  ├─ 列出了下位概念/构成要素（如"文化由符号、价值观、规范、物质构成"）
  │    → ✅ 严格按"亚概念关系"归特征
  │
  ├─ 未列下位概念，但用"第一…第二…"罗列了属性/特点
  │    → ✅ 放宽归特征
  │
  ├─ 原文明确说"XX具有以下特征/特点"
  │    → ✅ 归特征（无论内容是否涉及亚概念）
  │
  ├─ 是概念的测量指标/维度（如"人口规模的测量指标有…"）
  │    → ✅ 归特征
  │
  └─ 以上都不符合
       → ❌ 不归特征，检查是否属于其他要素，否则标⚠缺口
```

---

### ③ 外延（类型）

**标准**：概念内涵在现实中的分类展示。

**判定**：原文有"依据XX，可分为A/B/C"的完整结构 → 归外延。随口说"有很多种"但没展开 → 标缺口。

**注意**：类型和阶段的区别——如果分类隐含时间顺序（如"人口发展的三个阶段"），仍归外延，标题可以写"③外延（类型/阶段）"。

---

### ④ 功能

**标准**：概念/现象在社会运作中发挥的作用（功能主义视角）。

**方向**：**概念 → 社会**。"XX促进了/保证了/维系了/有利于…"

### ④ 原因与机制（补充维度）

一平哥模型未单独列出，但教材大量存在。

- **原因**：什么因素导致/影响 XX。方向是**因素 → 概念**，与功能相反。
- **机制**：从 A 到 B 的中间因果链条，怎么发生的。

**处理规则**：
- 原因和机制不强行拆分，统一标为"④原因与机制"
- 如果同一概念下同时有功能和原因/机制，分开写两个④板块（"④功能""④原因与机制"）
- 如果只有其中一个，就写一个

---

### ⑤ 视角

**标准**：不同学者/流派如何看待这个概念。

**判定**：必须有名有姓或明确理论流派标签（"XX认为…""XX学派指出…""从XX视角看…"）。无名无姓的泛泛观点不算视角。

**注意**：一个概念的定义可能同时来自某个学者（如泰勒对文化的定义），此时优先按原文使用位置归类——如果原文把它当定义引用就归①，如果原文把它当一个学术观点来讨论就归⑤。

---

### ⑥ 变迁

**标准**：概念在社会从传统到现代转型中的变化。

**判定**：必须有时态——有过去和现在的对比。"从…到…""经历了…阶段""传统上…而现代…"。当代才出现、没有历史演变内容的概念可以不标。

---

## 二、要素顺序规则

**强制固定顺序**：①内涵 → ②特征 → ③外延 → ④功能/原因与机制 → ⑤视角 → ⑥变迁

即使原文先讲变迁再讲定义，笔记中必须按标准顺序重新排列。理由：笔记是背诵主材料，不再对照课本，固定顺序便于跨章节对比和考场答题输出。

如果原文某个板块以某一要素为主、其他要素几乎没有，则只写该要素（保持固定序号），缺失的要素在末尾 ⚠ 中标注。

---

## 三、标记体系

| 标记 | 含义 | 使用位置 |
|------|------|---------|
| ★ | 核心句——答题可直接引用或转述的内容 | 仅用于**内容行**（`> ★ …`），不用于标题 |
| · | 衔接句——过渡内容、举例、背景铺垫 | 仅用于**内容行**（`> · …`） |
| ⚠ | 缺口——教材未覆盖/覆盖不足的要素维度 | 仅用于**板块末尾缺口列表** |
| ✏️ | 网课补充留白区 | 节末尾固定位置 |
| 💭 | 思考留白区 | 节末尾固定位置 |

**结构层级使用纯 Markdown 标题**（`##` `###` `####`），不用 ★ 做结构标记。

### ★ 核心句判定标准

标 ★（五类）：
1. 包含社会学专有术语的定义句
2. 列出特征/类型/功能的具体条目句（"第一…第二…第三…"）
3. 包含学者姓名+观点的句子
4. 包含因果关系或机制描述的句子
5. 包含时间维度变化（从A到B）的句子

不标 ★：
- 举例说明
- 纯数据/统计数字（定义性指标除外）
- 原文作者的口语化评论（"可以说""值得注意的是"等引导的）

## 四、要素覆盖与缺失处理

### 三级判断

| 情况 | 处理 |
|------|------|
| 教材有完整覆盖 | 正常标注 ★，归入对应要素 |
| 教材有相关内容但不独立/不完整 | 并入最近要素，标题中合并（如"③外延（类型/阶段）"），不加⚠ |
| 教材完全缺失 | 末尾⚠列出，注明缺失的要素名 |
| 教材缺失但该概念不需要该维度 | 不标⚠ |

### 各要素覆盖必要性

| 要素 | 覆盖要求 |
|------|---------|
| ①内涵、②特征 | **必须覆盖**（答题骨架），缺失必标⚠ |
| ③外延 | 概念有明确分类就必须覆盖；抽象单一概念可不标 |
| ④功能/原因与机制 | 解释性概念必须覆盖；纯描述性概念（如"人口密度"）可不标 |
| ⑤视角 | 至少一位学者观点才标；无学者讨论的概念可不标 |
| ⑥变迁 | 有历史演变内容必须覆盖；当代才出现的概念可不标 |

## 五、笔记结构模板

```markdown
## 第X节 XXXX

> · 原文导言：[节级总括句，保留原文逻辑脉络]

---

### 【核心概念A】整体定位

> · [该概念在全书/本章中的位置，原文概述]

#### ①内涵（定义）
> ★ 定义句

#### ②特征
> ★ 特征描述

#### ③外延（类型）
> ★ 分类

#### ④功能
> ★ 功能描述

#### ④原因与机制
> ★ 原因/机制描述

#### ⑤视角
> ★ 学者观点

#### ⑥变迁
> ★ 历史演变

#### ⚠ 缺口
- **XX要素**：说明

---

### 一、[教材自然板块标题]

[按同上结构重复]

---

✏️ 网课补充：（留白）
💭 你的思考：（留白）
```

### 板块划分规则
- 优先跟随教材的自然分节/分段标题
- 如果教材一节内无子标题，按概念单元自行划分
- 每个板块开头先写"整体定位"的 · 引文，再进入六要素

---

## 六、执行优先级

1. **忠实原文** > 框架完整：原文确实没有，宁可标⚠，不编造
2. **一平哥原意** > 个人理解：要素边界以本文件为准
3. **固定顺序** > 原文顺序：笔记中重新排列
4. **定义优先** > 特征优先：每个概念必须先有①再展开②
